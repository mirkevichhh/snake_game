Snake Game 🐍
This is a classic "Snake" arcade game implemented in Python using the Pygame library. The player must navigate a snake, collect apples, and avoid colliding with walls or the snake's own tail. The speed of the snake gradually increases with each apple consumed.

🛠 Requirements
Python 3.x

Pygame

🚀 Installation
Ensure you have Python installed.

Install the required Pygame library by running the following command in your terminal:

Bash
pip install pygame
▶️ How to Run
After installing the dependencies, navigate to the game folder and run the main file:

Bash
python main.py
🎮 Controls
The game features two distinct control schemes and an interactive menu system:

Movement:

You can use the Arrow keys (Up, Left, Down, Right) to navigate.

Alternatively, use the WASD keys (W for Up, A for Left, S for Down, D for Right).

Menu Navigation:

Use your Mouse to click the "Press to start the game" button in the main menu to begin your journey.

Pause & Resume:

Use the ESC key to pause the game at any time and resume when you are ready.

Game Over Options:

R — Restart the game instantly after losing.

Q — Quit the application.

🏆 Rules and Scoring
Objective: Collect apples to earn points (+10 points for each apple).

Difficulty: As your score increases, the snake's speed automatically grows, making maneuvering more challenging.

High Score:

The game automatically keeps track of your best performance.

Data Persistence: Your High Score is saved locally, so your best record will be waiting for you even after you close and restart the game.

Be careful—colliding with a wall or your own tail ends the game!
